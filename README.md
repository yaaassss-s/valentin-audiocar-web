# Valentín Audiocar

Web comercial responsive para Valentín Audiocar, local de venta e instalación de audio para vehículos en Wilde.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Desarrollo

- El sitio usa React, Vite y React Router.
- Las páginas están en `src/pages`.
- Los componentes reutilizables están en `src/components`.
- Los datos del negocio, navegación, servicios y galería están en `src/data`.
- Los placeholders de trabajos y hero tienen comentarios en el código para reemplazar por fotos reales.
- No hay horarios ni precios cargados porque todavía no fueron confirmados.
